# Nostra E-Commerce Website

Nostra is a modern, responsive e-commerce frontend project built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Optimized for different screen sizes, ensuring a seamless experience on mobile, tablet, and desktop.
- **Product Collections**: Browse different categories like Men, Women, and new arrivals.
- **Interactive UI**: Clean navigation, dynamic components, and smooth transitions.

## Project Structure

- `index.html` - The main homepage featuring new arrivals and brand details.
- `collections.html` - A dedicated page for viewing product collections.
- `contact.html` - Contact form for customer support.
- `style.css` - The global stylesheet powering the visuals.
- `script.js` - Contains the interactive logic for the website.

## Technologies Used

- **HTML5**: For semantic page structure.
- **CSS3**: For styling, layout, and responsive design.
- **JavaScript (Vanilla)**: For frontend interactivity.

## Getting Started

To view the project locally, simply open the `index.html` file in your preferred web browser. No complex build tools or servers are required.
