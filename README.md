This workspace contains several small JavaScript projects built with HTML, CSS, and vanilla JavaScript. Each project is self-contained and can be opened directly in a web browser.

## Projects

### `characterCounter`
- A real-time character counter application.
- Tracks typed characters and displays a warning when the maximum limit is reached.
- Files: `index.html`, `style.css`, `script.js`

### `nostraE-Commerce`
- A responsive e-commerce frontend project with homepage, collections, and contact pages.
- Contains product navigation and interactive UI elements.
- Files: `index.html`, `collections.html`, `contact.html`, `style.css`, `script.js`

### `registrationForm`
- A registration form with client-side validation.
- Validates name, email, and password before enabling submission.
- Files: `index.html`, `style.css`, `script.js`

### `studentlist`
- A student registration/list management form.
- Captures student details and shows a result table.
- Files: `index.html`

### `todolist`
- A simple to-do list application.
- Lets users add and manage tasks using JavaScript.
- Files: `index.html`, `script.js`

## Getting Started

1. Open the folder for the project you want to run.
2. Open the `index.html` file in your web browser.
3. No build tools or package installation are required.

## Notes

- Each project is designed for learning and experimentation with front-end web development.
- Feel free to modify and extend any project as needed.
